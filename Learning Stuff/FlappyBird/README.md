# Flappy Bird (Web + PWA)

This is a lightweight Flappy Bird clone built with plain HTML/CSS/JS + Canvas.

## Play

- Desktop: click or press **Space**
- Mobile: tap anywhere on the game

## Run locally

Any static server works. Examples:

```bash
python3 -m http.server 5173
```

Then open `http://localhost:5173/`.

## Install as an app (PWA)
